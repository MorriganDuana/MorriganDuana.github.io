What is NixOS

NixOS is a distribution based on the Linux kernel. It is recent since it is only dated 2013. Unlike the standard Linux distribution which uses a package manager such as apt or yum, NixOS is based on the Nix package manager.

This package manager can be found in the configuration.nix file, which also allows us to set up our entire system. We can compartmentalize the configurations and import them into the main configuration file. With a simple command (nixos-rebuils switch), our system has taken into account its configuration if there are no errors. The real plus here is that to create a new machine with the same configuration, just make a copy of the configuration.nix and put it on the new machine and that's it.

NixOS allows you to manage different configurations through versioning. If we break our system, all we have to do is reboot our machine and go back to a previous version that works. That's where NixOS gets its strength.
